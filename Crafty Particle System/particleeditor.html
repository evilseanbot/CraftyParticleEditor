Intro | Editor

This is a particle editor for CraftyJS.

To use in your CraftyJS game:

  1. Include the ParticleSystem.js file.
  2. Use the editor to create your particle system.
  3. Click on the Get JSON button and copy the JSON code into a file.
  4. Inside the game, make a ParticleSystem object and load your json file: [var system = Crafty.e("ParticleSystem").load("yoururl.json"); ]
  5. The ParticleSystem object has a 2D component and can be moved around, attached, etc, like any other 2D Crafty object.
